## Passing social link items

### As JSON string attributes

<Preview withToolbar>
  <Story name="with social link items" height="50px">
    {html`
      <jwc-layout-footer
        social-link-items='[{ "type": "github", "url": "https://github.com/jescacena" }]'
      ></jwc-layout-footer>
    `}
  </Story>
</Preview>

### As JSON object to the socialLinkItems property

```html
<script>
  var jwcFooter = document.querySelector("jwc-layout-footer");
  jwcFooter.socialLinkItems = [
    { type: "github", url: "https://github.com/jescacena" },
  ];
</script>
<jwc-layout-footer></jwc-layout-footer>
```
